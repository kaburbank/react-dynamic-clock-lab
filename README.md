# react-dynamic-clock-lab
SE04 Front End Development: React Lab

This clock was created using the following steps:
Install the project using Vite:
npm create vite@latest color-clock --template react
Type 'y' if prompted to install create-vite
Select 'React' as the framework
Select 'JavaScript' as the variant

Navigate to your project directory:
cd color-clock

npm install

npm install date-fns@2.30.0


The code for the app was developed in App.jsx using the format function from date-fns.

The app was styled using CSS

Use the command:
npm run dev
to run the app
